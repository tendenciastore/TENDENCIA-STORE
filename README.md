# Tendencia Store Argentina 🚀

¡Bienvenidos al repositorio oficial de Tendencia Store! 

Este proyecto contiene nuestro **Catálogo Interactivo de Ventas**, diseñado para ofrecer de forma rápida y visual nuestros productos de perfumería importada de diseñador, perfumes árabes, vapes y tecnología de punta.

## ✨ Características de la web:
* **Selector de Catálogo:** Switch rápido entre precios Minoristas ($ ARS) y Mayoristas (USD).
* **Pedido Directo:** Carrito de compras integrado que envía el pedido formateado directo a nuestro WhatsApp.
* **Diseño Moderno:** Interfaz rápida, urbana y adaptada a celulares.
